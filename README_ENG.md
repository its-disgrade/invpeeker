## Introduction

Inventory Peeker allows you to peek into another player's inventory.

And no, you can't steal anything. Look only. For now. (For now... Well... It depends on how many ideas you throw at me.)

## Controls

| Action | Key |
|--------|-----|
| Peek at nearest player | `U` |
| Peek at specific player | `SHIFT` + `Right Click` on player |
| Close inventory | `E` or `Esc` |

## Installation

1. Install [Fabric Loader](https://fabricmc.net/use/)
2. Download the mod from [Modrinth](https://modrinth.com/mod/lazyprogrammers-invpeeker) or [CurseForge](https://www.curseforge.com/minecraft/mc-mods/lazyprogrammers-invpeeker)
3. Place the `.jar` file in your `mods` folder
4. Launch the game

## FAQ

**Q: Can I steal items from other players?**  
A: No. The mod is read-only. You can only look.

**Q: Does it work on servers?**  
A: Yes, but the mod must be installed on **both** client and server.

**Q: Does it work with resource packs?**  
A: Yes, the mod uses vanilla GUI textures.
